A ORIGEM DO PROJETO

O MegaMan PET Mobile nasceu de um "sonho de consumo" de infância. Quem acompanhou a era de MegaMan NT Warrior lembra bem o que significava querer um Advanced PET — o dispositivo icônico que permitia aos operadores lutar ao lado de seus NetNavis.

Recentemente, ao adquirir o brinquedo original de 2004 (aquele que utilizava BattleChips físicos), a nostalgia se transformou em combustível para o desenvolvimento. Olhando para aquele hardware clássico, surgiu a ideia: "Por que não criar uma versão atualizada para os celulares de hoje?"

Este projeto é uma homenagem independente a esse legado, unindo o sentimento nostálgico com o desenvolvimento em Unity. O "coração" do jogo já está operante e pronto para ser testado!

NOTA DO DESENVOLVEDOR: O site e o jogo ainda estão em desenvolvimento. Nesta build de teste, os números de atributos estão inflados para facilitar a experimentação das mecânicas.

COMANDOS DE NAVEGAÇÃO

B

A

CONFIRMAR (Direito): Navega entre as telas de Status e Loja. Pressione para entrar no menu selecionado.

BOTÃO DE CONFIRMAÇÃO (DIREITO)

Na tela principal, utilize o Botão de Confirmação (Direito) para alternar entre os módulos: Status do MegaMan e Loja. Para acessar qualquer um desses menus, pressione o botão de confirmação novamente. Caso deseje retornar ao menu anterior ou à tela principal, utilize o Botão de Cancelar (Esquerdo).
